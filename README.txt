== Description ==

Tested on Wordpress 6.2

== Installation ==

1.-Download the last release as a zip file.
2.-In your Wordpress Dashboard, go to "Plugins" / "Add New".
3.-In the new page, click on "Upload Plugin".
4.-Select the zip file from your device and click "Install Now"
